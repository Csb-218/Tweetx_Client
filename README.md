
# TweetX

A social media app just like Twitter.


## Demo

https://tweetx-client.vercel.app

## Run Locally

Clone the project

```bash
  git clone https://github.com/Csb-218/Tweetx_Client.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
